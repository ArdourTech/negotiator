# Tasks

Development tasks for Negotiator

## deps
> Downloads and installs the necessary dependencies

~~~bash
lein deps
~~~

## test

> Runs the unit test suite

~~~bash
lein test
~~~

## install

> Builds and installs Negotiator to the local repository

~~~bash
lein install
~~~

## deploy

> Builds and deploys Negotiator to Clojars

~~~bash
lein deploy clojars
~~~
